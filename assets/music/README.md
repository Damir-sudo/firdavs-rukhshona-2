# Music

Place your background music file here and name it `background.mp3`.

The player will automatically use `assets/music/background.mp3` if it exists.
You can change the path in `js/script.js` inside the `WEDDING_CONFIG.music.src` field.

Recommended:
- Format: MP3
- Length: 2–5 minutes (it loops automatically)
- A soft, romantic instrumental track works best.
